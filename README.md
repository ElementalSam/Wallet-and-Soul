# Wallet and Soul

Wallet & Soul is a 2D platformer built in GameMaker Studio 2. It features split‑screen co‑op, diverse enemies, and player customization. It combines classic platformer mechanics with modern polish and scalability.

Motion mechanics: Jumping, dashing, attacking, and responsive movement.

Enemy variety: Patrollers, chasers, shooters, and terrain‑traversing AI.

Player state machine: Clean structure for managing player actions.

Controller support: Full input abstraction for both keyboard and gamepad.

Level generator: Dynamic and customizable stage layouts for replayability, alongside nearly two dozen handmade levels.

User settings: Fullscreen toggle, sound sliders, vertical/horizontal splitscreen, custom keybinds, and custom buttons.

Split‑screen co‑op: Two players can travel together with independent controls, items, and rewards.
